An expert system for Hospitals.

This medical-diagnosis expert system asks user for specific symptoms and gives possible disease that user may have. Also gives treatments for all symptoms.

To execute the program - 

1. Consult the file using [filename]. in swipl interpreter. In my case, file name is expert.pl hence using [expert]. to consult file.
2. Type in 'input.' to start the system.
3. Answer the questions in 'yes.' or 'no'.
4. Output will be given as diagnosis.
